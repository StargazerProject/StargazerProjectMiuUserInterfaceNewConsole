# StargazerProjectMiuUserInterfaceNewConsole
灵感来自 Azone 魔女っ子 水之魔女 Miu(美海) 的UI设计 增加了滚动条设计及最小化设计 增加了逐字符的控制台
原先的几代UI都是换行级别的日志控制台显示，这一次改为逐个字符显示，在文件下载过程中，可以正确的显示文件的下载进度
已经用于所有Stargazer系列的UI
